# Karas-Cipher

Karas-Cipher is an innovative symmetric cipher system based on the principles of "Stability Overflow" and reversible hash-based data transformation. Unlike standard algorithms, Karas-Cipher derives its security from a dynamic key fusion, combining a master hash with the binary signature of a specific file to create a highly non-linear, reversible encryption scheme.

## Key Features

- **True Reversibility:** Unlike traditional hash functions, Karas-Cipher is designed as a bi-directional symmetric cipher, allowing for precise decryption while maintaining hash-like diffusion properties.
- **Dynamic Diffusion:** The system utilizes a priority-based ROL (Rotate Left) mechanism. Constants (Hash + File Bytes) are sorted by magnitude; larger constants trigger faster, more impactful bit-rotations, ensuring every byte is cryptographically interdependent.
- **Stability Overflow:** Built to handle bit-level manipulations without breaking the decryption path, leveraging the "Stability Overflow" property to ensure data integrity.
- **Unicode Obfuscation:** Initial shifts in the Unicode table move plaintext out of standard frequency-analysis ranges, rendering statistical attacks ineffective.
- **Safe-Character Mapping:** Automatically maps non-printable hex characters back into the printable 32-126 range, ensuring the output remains transportable while remaining mathematically reversible.

## How It Works

1. **Fusion:** A composite key is generated by sorting the master hash and the file's raw binary signature.
2. **Shift:** Plaintext undergoes an initial Unicode shift.
3. **Rotation:** Bitwise rotations (ROL) are applied iteratively. The order of rotation is determined by the constant’s magnitude, creating a unique diffusion pattern per file instance.
4. **Normalization:** The resulting bytes are normalized into a printable character set, ensuring the cipher can be stored and transmitted easily without data corruption.

## Usage

Karas-Cipher is designed to integrate seamlessly into your Python projects.
to encode:
```python
import karas_cipher_encoder

# Plaintext to be encoded
text = "" # example: empty text

# Encrypting
encrypted = karas_cipher_encoder.karas_cipher_encrypt(text)
print(f"Encrypted output: {encrypted}")
```
to decode:
```python
import karas_cipher_encoder

# plaintext
text = "" # example: empty text
decrypted = karas_cipher_encoder.karas_cipher_decrypt(karas_cipher_encoder.karas_cipher_encrypt(text))
print(f"Decrypted Output: {decrypted}")
```
